Цель оценки.

Для каждой задачи каждый член команды независимо оценивал три параметра (в человеко-днях):

O - оптимистичная оценка (если всё идёт без серьёзных проблем);

M - наиболее вероятная оценка;

P - пессимистичная оценка (с учётом возможных задержек, рисков, неопределённости).

Индивидуальные оценки фиксировались в отдельных листах Google-таблицы. После этого для каждой задачи были вычислены индивидуальные ожидаемые длительности по формуле PERT:

$T_e = (O + 4·M + P) / 6$.
